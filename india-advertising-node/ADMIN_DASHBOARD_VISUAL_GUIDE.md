## 🎬 Interaction Patterns

### Button States

```
Normal:      [Button]
Hover:       [Button] ← Darker color, shadow
Active:      [Button] ← Different color
Disabled:    [Button] ← Grayed out
Loading:     [Button...] ← Spinning animation
```

### Form States

```
Empty:       ┌─────────────────────┐
             │ [Placeholder text] │
             └─────────────────────┘

Focused:     ┌─────────────────────┐
             │ [Cursor]           │ ← Blue border
             └─────────────────────┘

Filled:      ┌─────────────────────┐
             │ User entered text  │
             └─────────────────────┘

Error:       ┌─────────────────────┐
             │ User entered text  │ ← Red border
             └─────────────────────┘
             Error message here
```

### Alert States

```
Success Alert:  ✅ Data updated successfully!
                (Green background, auto-hide in 3s)

Error Alert:    ❌ Failed to save data
                (Red background, persistent)

Loading Alert:  ⏳ Saving...
                (Blue background)
```

## 🎯 User Flow Diagrams

### Add Product Flow

```
User clicks
"+Add Product"
    ↓
Form appears
    ↓
User fills form
    ↓
User clicks "Add"
    ↓
API request sent
    ↓
Success message ← OR ← Error message
    ↓
Form closes
    ↓
List updates
```

### Edit Product Flow

```
User clicks ✏️
    ↓
Form appears with data
    ↓
User modifies fields
    ↓
User clicks "Save"
    ↓
API request sent
    ↓
Success message
    ↓
Form closes
    ↓
List updates
```

### Delete Product Flow

```
User clicks 🗑️
    ↓
Confirmation dialog
    ↓
User confirms
    ↓
API request sent
    ↓
Success message
    ↓
Item removed from list
```
