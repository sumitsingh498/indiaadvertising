## 🔧 Troubleshooting

### Can't Login?

- Check password (default: `admin123`)
- Verify Node.js running
- Check `.env` file

### Changes Not Saving?

- Verify Node.js running on port 5000
- Check browser console (F12)
- Try refreshing page

### Images Not Showing?

- Check image URL format
- Verify files exist
- Try different image

### Dashboard Blank?

- Check console errors (F12)
- Verify servers running
- Clear browser cache
