## 🔧 Troubleshooting

### Can't Access Dashboard?

1. Check both servers running (ports 5000, 3000)
2. Clear browser cache
3. Try URL: `http://localhost:3000/indiaadvertising/admin`

### Wrong Password?

1. Default is `admin123`
2. Check `.env` file for correct password
3. Restart Node.js if you changed it

### Changes Not Saving?

1. Check Node.js running on port 5000
2. Open browser console (F12) for errors
3. Verify companyData.json is writable

### Images Not Showing?

1. Check image URL format (e.g., `/images/pic.jpg`)
2. Verify files exist in uploads folder
3. Check for CORS errors in console

👉 **More help**: See ADMIN_DASHBOARD_GUIDE.md
